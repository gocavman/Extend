# Animation Frames Needed

Create these animations in the 2D Stick Figure Editor. Each animation should be saved with the exact name listed below.

## Required Animations

### 1. **Stand** (Special - used when not moving)
- **Frames needed**: 1
- **Description**: Standing still pose

### 2. **Move** (Special - used when moving left/right)
- **Frames needed**: 4
- **Description**: Walking/running cycle
- **Note**: This replaces the old "guy_move" images

### 3. **Rest** (Level 1)
- **Frames needed**: 2
- **Description**: Resting/idle poses

### 4. **Run** (Level 2)
- **Frames needed**: 4
- **Description**: Running cycle (currently uses Move frames)
- **Note**: Already working, but you can customize if desired

### 5. **Jump** (Level 3)
- **Frames needed**: 3
- **Description**: Jumping animation sequence

### 6. **Jumpingjack** (Level 4 - Jumping Jacks)
- **Frames needed**: 4
- **Description**: Jumping jack poses

### 7. **Yoga** (Level 5)
- **Frames needed**: 8
- **Description**: Various yoga poses (downward dog, warrior, tree, etc.)

### 8. **Curls** (Level 6 - Bicep Curls)
- **Frames needed**: 4
- **Description**: Bicep curl animation (arms down, lifting, up, lowering)

### 9. **Kettlebell** (Level 7 - Kettlebell Swings)
- **Frames needed**: 8
- **Description**: Kettlebell swing motion

### 10. **Pushup** (Level 8 - Push Ups)
- **Frames needed**: 4
- **Description**: Push up motion (down, up, etc.)

### 11. **Pullup** (Level 9 - Pull Ups)
- **Frames needed**: 4
- **Description**: Pull up motion (hanging, pulling up, top, lowering)

### 12. **Meditation** (Level 10)
- **Frames needed**: 3
- **Description**: Meditation poses (sitting, breathing variations)

## How to Create

1. Open the 2D Stick Figure Editor
2. Create your pose
3. Save each frame with the exact animation name and frame number
   - Example: "Rest" with frame number 1, then "Rest" with frame number 2
4. The game will automatically load these frames when the action is performed
